# Copilot Agent — ЭТАП 1..3

В этой ветке реализованы три этапа в строгой последовательности:
- ЭТАП 1: статическая разметка и структура проекта
- ЭТАП 2: стилизация, клиентская валидация и UI-состояния
- ЭТАП 3: интеграция с API логина/регистрации и стабилизация

Разрешено для агента в этой ветке:
- Редактировать стили и клиентскую логику (в рамках ЭТАПОВ 2 и 3)
- Подключать вызовы API для login/register (только как описано)

Запрещено:
- Добавлять редиректы или навигацию
- Управлять пользовательской сессией
- Выполнять побочные эффекты за пределами UI
